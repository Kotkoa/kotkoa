# Project: kotkoa

GitHub profile repository for Andrey Kotko. Includes a GitHub Pages redirect page with Open Graph meta tags for social media previews.

## Structure

- `index.html` — OG meta tags + redirect to github.com/Kotkoa (hosted on GitHub Pages)
- `og-image.png` — Open Graph preview image (1200×630)
- `README.md` — GitHub profile README
- `settings.json` — VSCode editor settings (shared config)

## Hosting

- GitHub Pages: https://kotkoa.github.io/kotkoa/
- Redirects to: https://github.com/Kotkoa
- Used as shareable URL for LinkedIn/social media (GitHub blocks direct OG scraping)

## Code Style

- Indent: 2 spaces
- Quotes: single (JS/TS), double (HTML attributes)
- Formatter: Prettier (printWidth: 120)
- Final newline: yes
- Commit messages: conventional format (`feat:`, `fix:`, `chore:`, `style:`, `refactor:`, `test:`)
